Simple Image Gallery 

== Description ==

Faster and simpler than lightbox and nextgen. All you need to do is activate and let the plugin do its thing.

= Features =

*	Works on all clickable images in posts and pages.
*	No setting up needed.
*	Displays image title below the image.

= Developers plugin page =

[quick contact form plugin](http://quick-plugins.com/quick-image-gallery/).

== Installation ==

1.	Login to your wordpress dashboard.
3.	Go to 'Plugins', 'Add New' then search for 'Quick Image Gallery'.
4.	Follow the on screen instructions.
5.	Activate the plugin.

That's is. There is nothing else you need to do.
